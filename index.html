01правила

Четыре правила, которые нельзя нарушать.

Если хоть одно сломалось — ты уже не клоузер, а курьер сделок. Бизнес теряет деньги. Сначала эти четыре, потом всё остальное.

01

Не двигай сделки руками.

Закрой задачу с правильным результатом — Digital Pipeline сам передвинет сделку на нужный этап. Перетащил руками — убил аналитику и автоматизации.

02

У каждого касания — повод.

Не «напомнить про оплату», а конкретный повод: кейс, расчёт долга, юрист подготовил план. Тексты для MaxBot готовит копирайтер — твоё дело сделать звонок с поводом из таблицы.

03

Касание не реже раза в 3 рабочих дня.

На любом активном этапе если касаний нет 3+ рабочих дня — РОПу прилетает алерт. РОП вправе забрать клиента и передать другому клоузеру. Этап 6 (Отложен) — исключение: пауза до названной даты.

04

Zero Dashboard на конец дня.

К концу рабочего дня в воронке не должно быть просроченных задач. Делаешь скриншот чистого дашборда и отправляешь в чат с РОПом. Без скрина — день не закрыт.

ключевое слово

Касание — это исходящий звонок клиенту (любой результат, кроме «не пытался») или отправленное сообщение в MaxBot. Входящий звонок или сообщение от клиента сами по себе касанием не считаются — на них клоузер просто обязан ответить.

конец дня · zero dashboard

Ноль просрочек — ноль вопросов.

В 18:00 открываешь воронку, фильтруешь по своим сделкам, делаешь скрин: ни одной красной задачи. Скидываешь в группу с РОПом. Если есть просрочки — закрываешь, потом скрин. Без zero dashboard рабочий день не считается закрытым.

03этапы

По каждому этапу — что делать, к чему вести и куда уйдёт сделка.

Структура одна и та же: откуда сделка пришла → что делать → результат и переход → эскалация. Запомни шаблон один раз — будешь читать остальное по диагонали.

Встреча назначена

Сделка пришла горячая: клиент только что заплатил трипваер. Твоя единственная задача — провести встречу.

SLA · встреча в 24 часа максимум 2 переноса видео = Контур.Толк
Откуда приходит сделка

Автоматом из воронки МПП1, как только клиент оплатил трипваер. Охотник МПП1 проставил поле «Формат встречи» (видео или телефон) — от него зависит набор сообщений и автозвонков.

Что делать
  1. Открой задачу «Провести встречу с клиентом». Дедлайн = время встречи.
  2. Канал бери из поля «Формат встречи». Видео — Контур.Толк, телефон — обычный звонок.
  3. Если клиент переносит — поставь новую дату (макс. 2 переноса подряд).
  4. По итогу закрой задачу с одним из четырёх результатов ниже.

Догрев

Клиент пришёл на встречу, но «не готов прямо сейчас». Не отдавай горячее тепло — звонок через час, дальше касания каждые 3 дня.

обязательное поле · причина «не готов» 14 дней максимум звонок через 1 час обязателен
Что делать
  1. Заполни поле «Причина не готов» — без этого Pipeline не двигает сделку.
  2. Через 1 час после встречи — обязательный звонок. Клиент ещё «горячий», это лучшее окно дожать.
  3. Дальше клоузер сам выбирает день и время следующего касания, ориентируясь на пожелание клиента.
  4. Касание не реже раза в 3 рабочих дня — иначе РОП заберёт клиента.
  5. Если не дозвон — система переставит задачу автоматически (см. график ниже).
  6. День 7 — задача упадёт старшему клоузеру: «Попробуй вывести клиента на диалог».
  7. День 14 — финальная попытка. Без результата — автозакрытие с пометкой «неактивен».

Запрещено

  • Двигать сделки руками между этапами. Только закрытие задачи с результатом.
  • Касание без повода («просто напомнить про оплату»).
  • Писать тексты для MaxBot самому — их готовит копирайтер.
  • Третий перенос встречи на этапе 01 без эскалации РОПу.
  • Тишина 3+ рабочих дня на активных этапах. РОП заберёт клиента.
  • Зависать на этапе 07 больше 3 часов без передачи документов.
  • Закрыть рабочий день без zero dashboard в чате с РОПом.
  • Пропустить звонок через 1 час на этапе 03 (Догрев) и 04 (Счёт).

Обязательно

  • Закрывать каждую задачу с правильным результатом — Pipeline сам двигает сделку.
  • Заполнять обязательные поля до перехода: причина «не готов», дата возврата, сумма предоплаты.
  • Касаться клиента не реже раза в 3 рабочих дня на активных этапах.
  • Звонить через 1 час после встречи (Догрев) и после счёта (Счёт выставлен).
